## 添加新的协议适配器

### 1. 创建适配器文件

在 `src-tauri/src/protocol/adapters/` 下创建新文件：

```rust
// src-tauri/src/protocol/adapters/my_protocol.rs

use async_trait::async_trait;
use axum::http::HeaderMap;

use crate::models::Provider;
use crate::protocol::adapter::{ProtocolError, ProviderAdapter, UpstreamRequest, UpstreamResponse};
use crate::protocol::canonical::{CanonicalRequest, CanonicalResponse};

pub struct MyProtocolAdapter;

#[async_trait]
impl ProviderAdapter for MyProtocolAdapter {
    fn provider_type(&self) -> &'static str {
        "my_protocol"
    }

    async fn canonicalize_request(
        &self,
        req: &CanonicalRequest,
        provider: &Provider,
    ) -> Result<UpstreamRequest, ProtocolError> {
        // 1. 将 CanonicalRequest 转为你的协议格式
        // 2. 构建 headers（如认证头）
        // 3. 返回 UpstreamRequest

        let body = serde_json::json!({
            // 映射字段...
        });

        let mut headers = HeaderMap::new();
        // 添加认证头等

        Ok(UpstreamRequest {
            url: provider.api_base_url.clone(),
            headers,
            body,
        })
    }

    async fn parse_response(
        &self,
        resp: &UpstreamResponse,
    ) -> Result<CanonicalResponse, ProtocolError> {
        // 将上游响应转为 CanonicalResponse

        Ok(CanonicalResponse {
            id: resp.headers
                .get("x-request-id")
                .and_then(|v| v.to_str().ok())
                .unwrap_or("unknown")
                .to_string(),
            model: "unknown".to_string(),
            choices: vec![],
            usage: None,
        })
    }
}
```

### 2. 注册到管道

在 `src-tauri/src/gateway/pipeline.rs` 中添加：

```rust
use crate::gateway::middleware::my_middleware;

impl GatewayPipeline {
    async fn run_main(&self, ctx: RequestContext, body: Body) -> Result<RequestContext, StageError> {
        let ctx = extract::initialize(parts);
        let ctx = extract::read_body(ctx, body).await?;
        let ctx = resolve_route::run(&self.runtime, ctx).await?;
        let ctx = normalize_protocol::run(ctx).await?;
        let ctx = my_middleware::run(ctx).await?;  // 添加中间件
        let ctx = transform_request::run(ctx).await?;
        let ctx = dispatch_upstream::run(&self.runtime, ctx).await?;
        let ctx = transform_response::run(ctx).await?;
        Ok(ctx)
    }
}
```

```rust
pub struct MyModel {
    pub id: String,
    pub name: String,
    pub created_at: chrono::NaiveDateTime,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct NewMyModel {
    pub name: String,
}
```

### 2. 创建 Repository

在 `src-tauri/src/persistence/` 下创建：

```rust
// src-tauri/src/persistence/my_repo.rs

use sqlx::SqlitePool;
use crate::models::{MyModel, NewMyModel};

pub struct MyRepo;

impl MyRepo {
    pub async fn insert(pool: &SqlitePool, new: &NewMyModel) -> Result<MyModel, sqlx::Error> {
        let id = uuid::Uuid::new_v4().to_string();
        let now = chrono::Utc::now().naive_utc();

        sqlx::query_as!(
            MyModel,
            r#"
            INSERT INTO my_table (id, name, created_at)
            VALUES (?, ?, ?)
            RETURNING id, name, created_at
            "#,
            id,
            new.name,
            now
        )
        .fetch_one(pool)
        .await
    }

    pub async fn find_by_id(pool: &SqlitePool, id: &str) -> Result<Option<MyModel>, sqlx::Error> {
        sqlx::query_as!(
            MyModel,
            "SELECT id, name, created_at FROM my_table WHERE id = ?",
            id
        )
        .fetch_optional(pool)
        .await
    }
}
```

## 错误处理

### GatewayError

所有网关错误使用 `GatewayError` 枚举：

```rust
pub enum GatewayError {
    BadRequest(String),      // 400
    NotFound(String),        // 404
    Transform(String),       // 400 - 协议转换错误
    Upstream(reqwest::Error), // 502
    Timeout,                 // 504
    Database(sqlx::Error),   // 500
    Internal(String),        // 500
    Serialization(String),   // 500
}
```

## 代码规范

### Rust

- 使用 `thiserror` 定义错误枚举
- 使用 `serde` + `serde_json` 处理序列化
- 异步函数使用 `async fn`
- 错误处理使用 `Result<T, E>` 而非 panic

### 命名

- 文件名：`snake_case.rs`
- 结构体/枚举：`PascalCase`
- 函数/变量：`snake_case`
- 常量：`SCREAMING_SNAKE_CASE`

### 文档

- 公开函数添加 `///` 文档注释
- 复杂逻辑添加行内注释
- 使用中文注释（与项目约定一致）
